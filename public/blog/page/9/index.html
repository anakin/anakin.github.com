杯具啊



杯具, originally uploaded by anakin_sun.

俗话说人生就是茶几，上面摆满了杯具。
俗话又说人生就是茶叶，注定要泡在杯具里。
俗话老说实话，真讨厌。

例如今天，在天朝的互联网历史上就注定是杯具的一天，N个BT网站被封，very操蛋被和谐，虽然有微弱的声音跳出来说是服务器故障，大家少安毋躁，但是我们知道，和谐是早晚的事，默哀也不差这几天。

但是经过我深思熟虑，我认识到天朝这么做一定是为我们广大的网民着想，绝对是一切本着为网名服务的精神。流传于这些网站上的无数盗版的靡靡之音和宣扬资本主义思想的美剧已经毒害我们太久了，我们需要被拯救。要知道，美帝亡我之心不死，他们无时无刻不在搞和平演变，要警惕啊。

杯具都源于我们内心的纠结。纠结的原因是你忘记了你生活在天朝。

火炬之光



火炬之光, originally uploaded by anakin_sun.

这是我最近在玩的游戏。
据说是原来暴雪的人主力开发的，据说很多地方貌似《暗黑三》，据说因为它导致了《暗黑三》的跳票。我之前一致都很期待《暗黑三》的。
不管“据”是怎么说的，总之这款游戏还算值得一玩，玩过暗黑的同学们基本不用学习就能很快上手。有三种角色（法师，蛮子，弓箭手）可以选择，还可以带一个宠物（猫或者狗）。
游戏安装包只有400多M，效果还不赖，地图很深（地牢据说有120层，但是后面基本都是重复的，只是怪物的级别更高），怪物很凶猛，打着打着就觉得很累（用YC的话说，还不如去带小孩儿），随便一个小怪过来一刀就砍死你，唯一的前进途径就是不停的升级你的装备。
说到装备，和暗黑里面类似，但是种类更多，比如有护肩。同样分白色，蓝色，紫色（套装）和金色（unique）,而且可以不停的花钱在NPC那里强化，一直强化到很变态，只是有百分之几的机会会强化成白的，前功尽弃。
目前这个游戏还不算太完善，有一些著名的bug，比如之前的版本可以打劫商店（1.12修正了），目前经常还能碰到人物会卡在地图里等等。不过总的来说瑕不掩瑜吧，在《暗黑三》还没出现的日子里，填补一下空白吧。

最近看的几部电影



最近看的几部电影, originally uploaded by anakin_sun.

首先，照片是周末去神堂裕的时候拍的，和本主题没啥关系。
最近看了几部电影，唠叨一下。
《The.Boat.That.Rocked（海盗电台）》，讲一群疯狂的家伙在海上搞一个音乐电台的故事，情节不重要，欣赏的是他们自由，散漫，放纵和无所畏惧的那股劲儿，当然还有电影中不断让你惊喜的经典英伦摇滚，我看完了电影就赶紧把原声碟下载来丢到ipod里猛听了一阵儿，是不是忠实的摇滚乐迷不重要，好音乐谁都可以喜欢。
《Crank 2（怒火攻心）》，很久没有看到过这样从头烂到尾的烂片了，over。
《Music and Lyrics（K歌情人）》，休格兰特和《霹雳娇娃》里面那个白胖姑娘演的，情节简简单单，有点儿励志色彩，姑娘本是一个浇花的，但是张嘴就是歌词儿，终于通过自己的没怎么努力就写了一首相当不错的歌，演技派的格兰特大哥也因此由一个落魄的喜欢穿窄腿儿裤子的二手明星重新回到潮头浪尖儿，当然了，男的和女的最终幸福的搞在了一起。。。。。适合大家在周末闲的时候躺在洒满阳光的沙发上一边喝茶一边随便看看，里面的music还不错，我正听。
《斗牛》，黄渤和一头八路奶牛的故事。故事真实的反应了当年严重不靠谱的八路们的光荣行径。受伤的八路们在老乡家养伤，喝着荷兰来的一头奶牛的牛奶，村民们都不许喝。这个时候鬼子来了，八路要撤退，就是逃跑，当然了，肯定不能带着老百姓一起跑，这也就算了，还要把奶牛同志留给乡亲们代为照顾，奶牛啊，和村里的其他牛们长的完全不一样。这个光荣而艰巨的任务当然就落在黄渤身上了。我要说的是，这哥们儿演技越来越好了。

周末骑车去



周末骑车去, originally uploaded by anakin_sun.

9月12号，就是昨天，是周六。
一不小心参加了老丁叔组织的骑车出游活动，拖着我那辆一年没骑过的杂牌山地车，一圈下来，人基本上残废。
参与的人员除了图上看到的这7个，还包括一个正在拍照的小力，和中途逃走的大叔。
具体的路线是这样的，四惠－通县－顺义，再返回。由于我天然就住在通县，所以他们在四惠集合之后骑到风景秀丽的通县北关和我会合，然后一直骑到潮白河，沿着潮白河（说是沿着，我一路上也没怎么看见那个河）骑到顺义。由于走走停停，到了顺义已然是午饭时分，只好改变计划（原计划是去奥林匹克水上公园），在路边白吃老大爷地里西红柿的同时，打听到附近有一个叫做牌楼的地方，在当地最著名的“爱君饭店”大吃了一顿，所有的菜都是一样的配菜，当然也是一样的味道。休整之后，大叔由夫人开车接走，我们临时决定改变目的地，去顺义的汉石桥湿地，大概下午2～3左右到达，我只能说不枉我累死累活的骑这么远，风景确实相当可以。
”汉石桥湿地 位于杨镇和李遂镇交界处，占地约1.5万亩，茂密的芦苇荡和丰富的生物多样性，其中约有9700亩湿地位于杨镇境内，核心区芦苇荡面积有3000亩左右。 京郊平原独有的荒野景观“
进入湿地范围之后，道路很平整，两旁都是很高的树，湿地里满是荷花和芦苇以及各种不认识的草，中间还有各种不认识的鸟飞来飞去。人们坐在岸边闲聊打牌观鸟或者支个帐篷睡觉，惬意的很，我们旁边的帐篷里传出的鼾声吓我们一跳。
我们玩了一会儿，时间已经不早了，返回的时候直奔顺义县城，然后走通顺路奔通县，然后除了我之外的同学们再奔城里。
我要说的是，回程的路着实是对体力和毅力的双重考验。我们基本上是骑个七八公里就休息一会儿，屁股疼的没办法坐在车座上，腿也酸的根本没力气瞪，有几次真想打个taxi回家或者打电话叫家里人开车救援。尤其是中间有一段时间，看不到指向通县的路牌，也看不到希望。只是觉得一路都是上坡加顶风。机械的做蹬车的动作还能勉强对付凑合着前进，不小心碰到红灯需要停下来再接着走的时候，简直就绝望了。要是听前面的同学说走错了路，那简直就是噩耗加世界末日。
不管怎么说，天黑以后，晚上七八点钟的时候，我还是骑着车冲进了小区的门，所有的困难都再见，过程很美好。
经验是，以后要慎重参加类似的活动，呵呵。

秋天到了



IMG_1700, originally uploaded by anakin_sun.

听飞鱼秀说女的容易伤春，男的容易悲秋，我深以为然。
可是悲些什么呢？谁知道，显然也不重要。
北京的秋天还是很像个秋天的样子，天气变得凉凉的，穿一件长袖衣服盖着厚被子睡觉都觉得很温暖，我很喜欢这种感觉，盼望着不要再继续冷下去。阳光出来的时候，不像夏天那么浓烈刺眼，天很高，云也很淡，就差没看见过南飞雁。如果不是阴天下雨，到了下午，从办公室的窗户望出去就能看见西山还有晚霞和夕阳。我总觉得这样的时刻我们应该坐在城市最高的楼顶面向夕阳托腮遐想，或者在洒满阳光的院子里喝着茶看一只大狗跑来跑去。
从公司到家，夜里车少的时候也要开将近一个小时，一首歌的时间通常是三四分钟，可是为什么我觉得每次都听不了几首歌就到家了呢？

人生就是一次上线接着另一次上线



人生就是一次上线接着另一次上线, originally uploaded by anakin_sun.

又一次上线活动在这个周六晚上也就是这个周日的凌晨还算顺利的告一段落，这也意味着持续了几个星期的加班活动暂时可以停止一下下了。虽然领导同志们口口声声的保证以后不会再出现累死的拼命赶时间上线的情况，但是，领导永远都只是没事的时候保证，特殊情况永远都会准时在有事的时候出现。因为事实证明，不论时间多紧迫，到最后，任务总是会完成，领导总是会拍着胸脯去和领导的领导汇报说，经过我们的努力，搞定了。
所以，人生注定就是一次上线接着另一次上线。不足为怪。

说说加班的事吧。
其实一帮人一起加班这事说起来不是不能忍。通常晚上加班的时候效率比白天要好的多。不会有形形色色的人打扰，不会有领导开会，不会有菜市场一样的聒噪。夜深人静，夜凉如水，夜色正美。思路也很清晰，手指在键盘上乱飞，思绪比手指飞的还快。一帮人为了一个问题共同努力，一起研究或者一起吵架。投入的感觉很好。
更值得一提的，是半夜一个人开车回家。通常这个时候路上车已经比较少了，而且我觉得路上的绿灯亮的时候也要比红灯时间长一些。除了有几次，长安街晚上修路，变的七拐八拐的就像在开赛道（我没开过赛道，但我想，也就这个感觉吧）。路灯昏黄，车里很暗，也很安静，静静的听eason卖力的唱着，一首又一首，他唱的那么努力，那么认真，那么让人感动。有几次我都想时间是否可以过的慢点儿，再慢点儿，或者干脆就停在这一刻好了。

我们在追求些什么呢？或许我们追求的那些，已经在身边了。

What Is Cocoa?

Cocoa is a complete set of classes and application programming interfaces (APIs) for building Mac OS X
applications and tools. With over 240 classes, Cocoa is divided into two essential frameworks: the
Foundation framework and the Application Kit.
The Foundation framework provides a fundamental set of tools for representing fundamental data types,
accessing operating system services, threading, messaging, and more. The Application Kit provides the
functionality to build graphical user interfaces (GUI) for Cocoa applications. It provides access to the
standard Aqua interface components ranging from menus, buttons, and text fields—the building blocks of
larger interfaces—to complete, prepackaged interfaces for print dialogs, file operation dialogs, and alert
dialogs. The Application Kit also provides higher-level functionality to implement multiple document
applications, text handling, and graphics.
Classes are not the only constituents of the Cocoa frameworks. Some programming tasks, such as
sounding a system beep, are best accomplished with a simple C function. Cocoa includes a number of
functions for accomplishing tasks such as manipulating byte orders and drawing simple graphics.
Additionally, Cocoa defines a number of custom data types and constants to provide a higher degree of
abstraction to many method parameters.
